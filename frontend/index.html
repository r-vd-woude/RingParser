RingParser

Map files to XSD schema for Griel Bulkupload

Step 1: Upload File

Drag and drop your file here

Step 2: Map Columns to Schema

Select the target fields from the XSD schema and map them to the columns in your uploaded file.

Fields in red with an * are required while white fields are optional. You can also add biometric data if you have that.

The mapper will automatically detect names similar to the target fields and suggest mappings, but you can change them manually if needed. On the right side of mapping interface, you will see a button with "✓ View sample". Clicking that will show you what data is in the first few rows of the selected column.

Biometrics

Step 3: Validate Data

Validate your mapping against the XSD schema. This will check for required fields, data types, and any inconsistencies.

Validation results will appear here

Step 4: Generate XML

Once your mapping is validated, you can generate the XML file. You can preview the generated XML before downloading it.

If you uploaded a file with more than 1000 records, multiple XML files will be generated. Each file will contain up to 1000 records. You can download all generated XML files as a ZIP archive.
